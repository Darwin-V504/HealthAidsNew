Herramientas necesarias:
-VS Code
-Node.js versión +18 en adelante
-Expo go SDK 54 (App Store/PlayStore)
-Comprensión Lectora
-Paciencia
-Cerebro
Manual para ejecutar:
Descarga el proyecto como arzhivo zip
Extrae todo el archivo zip
Abres VS Code
Click en File y abres la carpeta extraída ubicada en tu explorador de archivos (usualmente situada en descargas)
Una vez que estas dentro del proyecto, abres terminal y escribes "npm install" para descargar todas las dependencias y librerías a utilizar
Siempre en terminal, escribes "npx expo start" comando para correr el proyecto 
Con tú teléfono (Android-IOS) abres Expo Go App y escaneas el codigo qr que apareció en terminal
Asegurate de que se vea una pantalla de inicio y que el backend ya este corriendo
